## Project Setup

### Install ESLint globally using

```bash
npm install -g eslint
```

### clone the repository and run

```bash
cd nanoplayer-vue/
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run serve
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
